# Sentence Analyzer

Uses JavaScript loops to build a sentence analyzer.

## UX

**Getting Started**

In the input field, key in the text (e.g. Hello World).  You will see the number of vowels, consonants, punctuation and words.

**User Stories**

- As a user, I should be able to key in text
- As a user, I should be able to get the number of vowels
- As a user, I should be able to get the number of consonants
- As a user, I should be able to get the number of punctuation characters
- As a user, I should be able to get the number of words
- As a user, if I click on the Clear button, I expect the information to be cleared

## Features

Counts the number of vowels, consonants, punctuation characters and words.

## Technologies

Uses HTML5, CSS3, JavaScript, jQuery 3.7.1 and Bootstrap 5.3.2.
